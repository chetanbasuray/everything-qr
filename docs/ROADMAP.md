# QR Studio Roadmap

This roadmap lists every planned or potential capability for a full-spectrum QR studio. Each item can become its own GitHub issue.

## Core generator

- Add dynamic sizing presets (badge, sticker, poster, signage)
- Add SVG export
- Add PDF export
- Add EPS export
- Add multi-format download bundles
- Add high-DPI export (2x, 3x)
- Add transparent background export
- Add color palette presets
- Add brand kit presets
- Add custom logo overlay with safe zones
- Add rounded modules and eye styling
- Add gradient fills and duotone themes
- Add background textures and patterns
- Add QR readability testing and warnings
- Add live payload validation and formatting hints
- Add payload length meter with capacity warnings
- Add auto-correct for URLs (missing scheme)
- Add payload templates for common use cases
- Add QR code version selector
- Add custom quiet zone controls
- Add multi-QR canvas layout (print sheets)
- Add smart label generation (name + date)
- Add batch QR generation from CSV
- Add API-based bulk generation

## Payload coverage

- Add vCard 4.0 support
- Add MECARD contact support
- Add Apple Wallet pass links
- Add Google Wallet links
- Add UPI payment payloads
- Add SEPA payment payloads
- Add PayPal and Stripe payment links
- Add WhatsApp Business chat links
- Add Telegram and Signal deep links
- Add Messenger deep links
- Add Instagram profile payloads
- Add LinkedIn profile payloads
- Add YouTube channel links
- Add App clip / Instant app links
- Add Wi-Fi WPA3 fields and hidden network toggles
- Add email multiple recipients support
- Add calendar recurrence rules (RRULE)
- Add location with map provider selector
- Add event timezone selection
- Add SMS with character count and MMS
- Add crypto network presets (USDT, SOL, etc.)
- Add custom payload builder for advanced users
